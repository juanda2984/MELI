# Proyecto Star Wars para MELI
**Este proyecto se realiza con finalidades educativas y de demsotración**

El proyecto fue realizado completamente en Java usando SpringBoot con librerias de Spring Rest para el uso de servicios rest en formato Json,
luego fue desplegado en un servidor Cloud llamado heroku. Adicionalmente el proyecto esta completamente depositado en GitHub tal como se soliicitaba
en los requerimientos no funcionales de la prueba.

El objetivo de la aplicacion es realizar una traingulacion de un punto dado, teniendo en cuenta otrso tres con sus respectivas coordenadas y la distancia de cada
uno de estos ultimos tres con el punto incognito y de esta forma conocer las coordenadas del punto incognito, para este punto se uso una libreria llamada trilateration que tiene un algoritmo de trilateración usado comunmente en gps; adicional el requerimiento funcional de la prueba solicitaba realizar el control de un mensaje enviado ´por cada punto y conjugarlo en uno solo.

para la respectiva validación de la prueba se requiere descomprimir el archivo test.zip e importar en SoapUi el proyecto, luego simplemente ejecutar cada uno de los request segun los nieveles de la prueba
